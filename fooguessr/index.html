FooGuessr

A browser-based game where you guess the programming language of a random snippet from Github.

ROUND x OF x:

Loading code block...

Languages were chosen on a highly selective basis, i.e whichever languages in GitHub's Linguist system I have myself heard of at some point for any reason. Please do not ask me to include Malbolge or whatever esolang conceived by programmers in the Siberian wilderness you personally champion. (Brain▒▙█ is an exception because we love Brain▟▒▀).
